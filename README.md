    <article class="markdown-body entry-content" itemprop="text"><h1><a id="user-content-technical-assignment-front-end-engineer" class="anchor" aria-hidden="true" href="#technical-assignment-front-end-engineer"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Technical assignment front-end engineer</h1>
<p>Make an application that allows the user to view the observed (in the past 30 days) or forecasted (in the future) daily weather conditions for a given location using the <a href="https://darksky.net/dev/docs" rel="nofollow">Dark Sky API</a>.</p>
<p>You are free to use the tools and frameworks you prefer unless stated otherwise.</p>
<p><strong>Requirements</strong></p>
<p>The three pillars of front-end development are HTML, CSS and JavaScript. We would like to see you know how to use them for the right job.</p>
<ul>
<li>Use ReactJS</li>
<li>Pay attention to semantics: any HTML might work, but choosing the right tags show you care</li>
<li>Show us you know how to create a modern user interface using CSS or your favourite CSS preprocessor</li>
<li>Communicate to the Dark Sky API through a PHP back-end</li>
</ul>
<p><strong>Timing</strong></p>
<p>We don't specify a timeframe for the assignment. Just ping us when you are done, on this endpoint (<a href="https://api.slack.com/incoming-webhooks" rel="nofollow">Slack incoming webhooks</a>): <code>https://hooks.slack.com/services/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</code>. Please mention your name and a link to what we need to review.</p>

# Technical assignment front-end engineer

Make an application that allows the user to view the observed (in the past 30 days) or forecasted (in the future) daily weather conditions for a given location using the [Dark Sky API](https://darksky.net/dev/docs).

You are free to use the tools and frameworks you prefer unless stated otherwise.

**Requirements**

The three pillars of front-end development are HTML, CSS and JavaScript. We would like to see you know how to use them for the right job.

* Use ReactJS
* Pay attention to semantics: any HTML might work, but choosing the right tags show you care
* Show us you know how to create a modern user interface using CSS or your favourite CSS preprocessor
* Communicate to the Dark Sky API through a PHP back-end



This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
